## 5. 盈亏平衡分析

### 5.1 盈亏平衡点计算

```
月固定成本 = ¥2,800（M1-3）/ ¥12,800（M7-12）
变动成本率 = 20%
单位贡献毛利 = ¥3,999 × (1 - 20%) = ¥3,199

盈亏平衡用户数 = 固定成本 / 单位贡献毛利
M1-3: ¥2,800 / ¥3,199 = 0.9 个用户/月 → 1个用户即盈利
M7-12: ¥12,800 / ¥3,199 = 4 个用户/月 → 4个用户即盈利
```

**结论：产品从第一个月起即可盈利（创始人不计薪酬），即使计入人力成本，月均4个新用户即可覆盖全部成本。**

## 8. 增长飞轮与规模效应

### 8.1 飞轮模型

```
更多付费用户 → 更多案例和证言 → 更强的社会证明
    → 更高的转化率 → 更低的CAC → 更多利润
    → 更多内容投入 → 更多曝光 → 更多付费用户
```

### 8.2 规模效应

- **零边际成本**：每新增一个用户的交付成本接近于零
- **案例积累**：用户越多，案例库越丰富，产品价值越高
- **代理商网络**：代理商越多，触达范围越广，获客效率越高
- **Prompt优化**：用户反馈越多，系统Prompt越精准，输出质量越高
