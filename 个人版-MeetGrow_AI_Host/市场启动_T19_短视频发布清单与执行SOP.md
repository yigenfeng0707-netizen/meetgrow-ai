## 4. 拍摄设备清单和设置建议

### 4.1 基础设备清单（手机级别）

| 设备 | 推荐型号/规格 | 预算 | 用途 |
|------|-------------|------|------|
| **手机** | iPhone 13及以上 或 同级别安卓（华为/小米旗舰） | 已有即可 | 主力拍摄设备 |
| **手机稳定器** | DJI OM系列 / 影石Flow | 300-900元 | 消除画面抖动，提升专业感 |
| **领夹麦克风** | 博雅BY-WM4 Pro / Rode Wireless Go II | 150-800元 | 人声清晰是视频质量关键 |
| **补光灯** | 环形补光灯（14寸以上） | 50-200元 | 面部光线均匀，画面质感提升 |
| **手机支架** | 桌面+落地各一个 | 30-80元 | 固定拍摄和屏幕录制 |
| **背景布/板** | 纯色（白/灰/深蓝） | 30-50元 | 干净背景提升专业度 |

**总预算：560-2,030元（不含手机）**

### 4.2 手机拍摄设置

#### iPhone 设置

```
相机 > 设置：
- 分辨率：4K / 30fps（平衡画质和文件大小）
- 格式：高效（HEVC），兼容性优先则选"兼容性最佳"
- 网格线：开启（辅助构图）
- 稳定模式：标准（上稳定器后可关闭）
- HDR：关闭（后期剪辑兼容性更好）
- 对焦/曝光锁定：长按屏幕锁定
```

#### 安卓手机通用设置

```
相机 > 设置：
- 分辨率：1080P / 30fps（兼容性最好）或 4K / 30fps
- 视频编码：H.265（省空间）或 H.264（兼容性好）
- 防抖：开启电子防抖
- 对焦模式：连续自动对焦
- 白平衡：手动锁定（避免色温跳变）
```

### 4.3 拍摄构图技巧

| 技巧 | 说明 | 适用场景 |
|------|------|---------|
| **三分法** | 人物放在三分之一处，不要正中 | 所有口播场景 |
| **留白** | 画面上方或侧方留出文字空间 | 后期需要加字幕/贴纸 |
| **竖屏9:16** | 抖音/视频号/小红书必须竖屏 | 短视频主力平台 |
| **横屏16:9** | B站深度内容可用横屏 | B站长视频 |
| **背景简洁** | 纯色墙面或书架，避免杂乱 | 所有场景 |

### 4.4 收音注意事项

- 室内拍摄时关闭空调、风扇等噪音源
- 领夹麦距离嘴巴15-20cm为最佳
- 拍摄前录制5秒环境音，后期可用于降噪参考
- 避免在空旷大房间录制（回声严重），可选择有软装的房间

## 5. 屏幕录制教程

### 5.1 录制工具选择

| 工具 | 平台 | 费用 | 推荐指数 |
|------|------|------|---------|

### 5.2 OBS Studio 录制设置（推荐方案）

#### 安装与基础设置

```
1. 下载安装 OBS Studio：https://obsproject.com
2. 新建场景："MeetGrow AI 演示"
3. 添加来源：
   - 来源1：显示器采集（全屏录制）或 窗口采集（指定浏览器窗口）
   - 来源2：音频输出采集（系统声音）
   - 来源3（可选）：视频采集设备（摄像头，用于画中画）
```

#### 录制参数设置

```
设置 > 输出 > 录制：
- 录制格式：mkv（录制完成后可转mp4）
- 编码器：硬件编码（NVIDIA NVENC / AMD / Intel）
- 码率：6000-8000 Kbps（1080P）或 4000-6000 Kbps（720P）
- 关键帧间隔：2秒

设置 > 视频：
- 基础分辨率：1920x1080
- 输出分辨率：1920x1080（不缩放）
- 帧率：30fps
```

#### 竖屏录制设置（短视频平台）

```
设置 > 视频：
- 基础分辨率：1080x1920（手动输入）
- 输出分辨率：1080x1920
- 或者：手机投屏到电脑后，用窗口采集手机画面
```

### 5.3 产品演示录制流程

#### 录制前准备（Checklist）

#### 录制中注意事项

- 鼠标移动要**慢而稳**，不要快速乱晃

- 关键步骤配合**口播讲解**
- 出错不要停，**暂停录制**调整后继续
- 每个功能演示之间留**2秒静帧**（方便剪辑）

## 6. 剪辑工作流

### 6.1 推荐剪辑工具

| 工具 | 平台 | 费用 | 适合场景 |
|------|------|------|---------|
| **剪映** | 手机+电脑 | 免费 | 首选，模板多、自动字幕、一键发布 |
| **CapCut（剪映国际版）** | 全平台 | 免费 | 同剪映，界面更简洁 |
| **必剪** | 手机+电脑 | 免费 | B站投稿优化，B站特色贴纸 |
| **Premiere Pro** | 电脑 | 付费 | 复杂剪辑需求 |
| **Final Cut Pro** | Mac | 买断 | Mac用户专业剪辑 |

**推荐方案：剪映（主力） + 必剪（B站投稿）**

### 6.2 完整剪辑流程（从录制到发布）

```
步骤1：素材整理（10分钟）
  ├── 将录制视频传输到电脑/手机
  ├── 按"日期_视频编号_内容"命名文件
  ├── 删除废片，保留可用片段
  └── 准备好Logo、产品截图等辅助素材

步骤2：粗剪（20-30分钟）
  ├── 导入剪映/CapCut
  ├── 按脚本顺序排列片段
  ├── 剪掉停顿、口误、重复部分
  ├── 控制总时长在目标范围内
  └── 添加转场（推荐：淡入淡出、推拉，不要太花哨）

步骤3：精剪（20-30分钟）
  ├── 添加字幕（剪映自动识别+手动校对）
  ├── 关键信息添加文字强调（放大、变色）
  ├── 添加箭头/圈注/高亮（指引观众视线）
  ├── 调整节奏：重点部分放慢，过渡部分加速
  └── 添加BGM（音量控制在人声的20-30%）

步骤4：包装（15分钟）
  ├── 添加片头（3-5s品牌标识+产品名）
  ├── 添加片尾（CTA：关注+联系方式+体验链接）
  ├── 添加角标Logo水印
  ├── 制作封面图
  └── 整体预览，检查错别字和画面

步骤5：导出（5分钟）
  ├── 分辨率：1080x1920（竖屏）或 1920x1080（横屏）
  ├── 帧率：30fps
  ├── 码率：推荐8Mbps以上
  ├── 格式：MP4（H.264）
  └── 文件大小控制在500MB以内

步骤6：发布（10分钟）
  ├── 撰写标题、描述、标签
  ├── 上传封面图
  ├── 设置定时发布（或即时发布）
  ├── 发布后立即检查播放效果
  └── 转发到微信群和朋友圈
```

## 10. 爆款视频特征分析

### 10.1 短视频爆款公式

```
爆款 = 强钩子（前3秒） + 信息密度（中间） + 行动召唤（结尾）
```

#### 前3秒钩子模板

| 钩子类型 | 模板 | 示例 |
|---------|------|------|
| **痛点共鸣** | "还在为XX发愁？" | "还在为展会招不到商发愁？" |
| **反常识** | "XX其实是错的" | "招商靠人脉？其实大错特错" |
| **数字冲击** | "X天/X倍/X%" | "3分钟搞定一周的招商工作量" |
| **悬念设置** | "我发现了一个XX" | "我发现了一个会展人都在偷偷用的工具" |
| **身份锚定** | "如果你是XX，一定要看" | "如果你是会展公司的老板，这条一定要看完" |
| **对比反差** | "XX vs XX" | "月薪8000的招商专员 vs 年费3999的AI" |
| **紧迫感** | "再不知道就晚了" | "2026年了还不用AI招商，你会被淘汰" |

### 10.2 高完播率视频特征

| 特征 | 说明 | 操作方法 |
|------|------|---------|
| **开头即高潮** | 前3秒展示结果或抛出悬念 | 先展示AI生成的方案成果，再演示过程 |
| **节奏紧凑** | 每5秒一个信息点或画面变化 | 剪辑时删除所有"嗯""啊"和停顿 |
| **视觉变化** | 画面切换、文字弹出、缩放 | 每3-5秒切一次画面角度或添加元素 |
| **字幕强调** | 关键信息用大字突出 | 剪映关键帧放大+变色 |
| **故事线** | 有起承转合的叙事 | "问题 > 尝试 > 发现 > 结果 > 推荐" |
| **情绪共鸣** | 引发认同、惊讶、好奇 | 说出观众心里话，展示惊人效率 |
| **长度适中** | 不拖沓，信息密度高 | 宁可短10秒，不要多10秒废内容 |

### 10.3 高互动率视频特征

| 特征 | 说明 | 操作方法 |
|------|------|---------|
| **开放式问题** | 结尾抛出问题引导评论 | "你们公司招商最头疼哪个环节？" |
| **争议性观点** | 表达鲜明立场引发讨论 | "传统招商模式已经死了，你同意吗？" |
| **投票/选择** | 给观众二选一 | "你觉得AI和人工谁更强？评论区选一个" |
| **留白/不说完** | 留悬念让人追问 | "还有一个隐藏功能，评论区告诉我再出教程" |
| **评论区互动** | 主动回复并追问 | 回复评论时反问，拉长对话链 |
| **置顶评论引导** | 自己发第一条评论 | "想要体验链接的扣1，我私信发你" |

### 10.4 会展行业短视频爆款选题方向

| 选题方向 | 热度 | 竞争度 | 示例标题 |
|---------|------|--------|---------|
| **AI工具测评** | 极高 | 中 | "5款AI招商工具横评，最后一个绝了" |
| **行业痛点吐槽** | 高 | 低 | "会展招商最难的是什么？评论区炸了" |
| **效率对比** | 高 | 低 | "同事加班3天的方案，我10分钟就搞定了" |
| **行业内幕** | 高 | 中 | "会展行业不会告诉你的5个招商潜规则" |
| **工具推荐** | 中高 | 高 | "会展人必备的3个效率神器" |
| **案例拆解** | 中 | 低 | "这场展会招商300家的秘密武器" |
| **趋势分析** | 中 | 低 | "2026年会展行业3大趋势，AI排第一" |
| **薪资/收入** | 高 | 中 | "会展招商月薪3万的人都在用什么方法" |

### 10.5 避免踩坑清单

| 不要做的事 | 原因 | 正确做法 |
|-----------|------|---------|
| 开头自我介绍 | 浪费前3秒，用户划走 | 开头直接抛钩子，简介里写身份 |
| 画面模糊/暗 | 不专业，完播率低 | 充足光线+稳定画面 |
| 念稿明显 | 不自然，缺乏信任感 | 记要点，自然表达 |
| BGM盖过人声 | 听不清说什么 | BGM音量控制在人声的20-30% |
| 没有字幕 | 很多人静音刷视频 | 必须加字幕，剪映一键生成 |
| 硬广无内容 | 没人看纯广告 | 70%干货+30%产品植入 |
| 全平台发一样的 | 各平台调性不同 | 根据平台特点做差异化适配 |
| 发完不管 | 不互动=放弃推荐量 | 发布后1h内密集互动 |
| 急于求成 | 前几条数据差很正常 | 坚持发满20条再做判断 |
| 忽视评论区 | 评论区是二次传播金矿 | 每条评论都认真回复 |

### 10.6 爆款视频复制策略

当某条视频数据明显高于平均时，执行以下"爆款复制"流程：

```
步骤1：分析爆款因素
  - 是标题钩子好？还是内容选题好？还是发布时间好？
  - 提取可复用的核心元素

步骤2：制作同系列视频
  - 同一选题的不同角度（如换一种钩子）
  - 同一钩子的不同选题
  - 爆款视频的续集/深度版

步骤3：跨平台翻拍
  - 在数据好的平台发深度版
  - 在还没发的平台发适配版

步骤4：追投DOU+/付费推广
  - 数据好的视频值得追投
  - 100元小额测试，ROI>1则加码

步骤5：引流到私域
  - 爆款视频带来大量曝光时，评论区/私信引导加微信
  - 准备好承接话术和体验链接
```
