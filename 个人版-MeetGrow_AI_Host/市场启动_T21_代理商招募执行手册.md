## 一、代理商画像定义

### 1.1 理想代理商画像

MeetGrow AI 的理想代理商/渠道合伙人通常具备以下特征中的至少3项：

**行业资源型**：
- 会展行业从业者（3年以上经验），拥有会展主办方、场馆方人脉
- 商协会工作人员，日常接触大量有招商需求的企业
- 政府招商部门相关人员或离职创业者

**销售能力型**：
- 有SaaS/软件销售经验，熟悉订阅制产品的销售逻辑
- 擅长微信私域运营，有500人以上的行业社群
- 有过招商/营销类产品的推广经历

**内容影响力型**：
- 会展行业自媒体/KOL（公众号、抖音、知乎有一定粉丝基础）
- 行业培训机构讲师，定期给会展从业者授课
- 会展行业社群群主或活跃意见领袖

**技术赋能型**：
- 为会展公司提供搭建、设计、技术服务的供应商
- 已有客户基础，可以将MeetGrow AI作为增值服务推荐
- 数字营销/活动策划公司的业务负责人

### 1.2 代理商来源优先级

| 优先级 | 来源渠道 | 预期转化率 | 开发难度 |
|--------|----------|-----------|---------|
| P0 | 种子用户中表现活跃者 | 40% | 低 |
| P1 | 行业社群群主/KOL | 25% | 中 |
| P2 | 会展服务商/搭建商 | 20% | 中 |
| P3 | SaaS软件代理商 | 15% | 高 |
| P4 | 冷启动招募（招聘平台） | 5% | 高 |

### 1.3 非理想代理商特征（不建议合作）

- 没有行业资源，纯靠朋友圈推广
- 追求短期暴利，不愿投入时间学习产品
- 同时代理5个以上竞品，无法专注
- 沟通成本高，需要手把手教基础操作
- 对产品价值缺乏认同，只是"试试看"心态

## 二、代理商分级体系详解

### 2.1 四级体系总览

```
铜牌 → 银牌 → 金牌 → 钻石
15%     20%     25%     30%
```

### 2.2 各级别详细标准

#### 铜牌代理（入门级）

| 项目 | 标准 |
|------|------|
| 分润比例 | 15%（每单约¥600） |
| 入门门槛 | 购买1份产品自用（¥3,999） |
| 月度考核 | 无硬性要求 |
| 季度考核 | 至少推荐1个付费客户 |
| 权益 | 销售工具包、基础培训、专属推广码 |
| 义务 | 正确使用产品、维护品牌形象 |
| 适合人群 | 兼职推广、资源试水 |

#### 银牌代理（进阶级）

| 项目 | 标准 |
|------|------|
| 分润比例 | 20%（每单约¥800） |
| 升级条件 | 累计推荐5个付费客户 |
| 月度考核 | 至少推荐1个新客户 |
| 季度考核 | 季度销售额≥¥15,000（约4单） |
| 权益 | 铜牌全部权益 + 1V1销售辅导、优先新模板体验、月度线上培训 |
| 义务 | 每月至少1次客户Demo、反馈市场信息 |
| 适合人群 | 有一定资源、稳定推广 |

#### 金牌代理（核心级）

| 项目 | 标准 |
|------|------|
| 分润比例 | 25%（每单约¥1,000） |
| 升级条件 | 累计推荐15个付费客户 |
| 月度考核 | 至少推荐2个新客户 |
| 季度考核 | 季度销售额≥¥40,000（约10单） |
| 权益 | 银牌全部权益 + 线下培训名额、定制营销物料、联合品牌活动、客户线索分配 |
| 义务 | 每月至少2次客户Demo、协助内容创作、维护区域客户关系 |
| 适合人群 | 全职投入、有团队支撑 |

#### 钻石代理（战略级）

| 项目 | 标准 |
|------|------|
| 分润比例 | 30%（每单约¥1,200） |
| 升级条件 | 累计推荐30个付费客户 + 面谈考核通过 |
| 月度考核 | 至少推荐3个新客户 |
| 季度考核 | 季度销售额≥¥80,000（约20单） |
| 权益 | 金牌全部权益 + 区域独家保护、产品功能投票权、年度峰会VIP席位、股权激励机会 |
| 义务 | 区域市场开发、团队管理、品牌共建 |
| 适合人群 | 创业型合伙人、区域总代 |

### 2.3 升降级规则

**升级路径**：
- 铜牌→银牌：累计5个付费客户，且最近3个月有持续出单
- 银牌→金牌：累计15个付费客户，且最近季度达标
- 金牌→钻石：累计30个付费客户 + 面谈评估通过

**降级保护**：
- 连续2个季度未达标 → 预警通知
- 连续3个季度未达标 → 降一级
- 降级后重新达标可恢复原级别

**特别说明**：
- 首签代理商默认铜牌，有资源的可申请银牌（需提供资源证明）
- 所有代理商享受3个月新手保护期，保护期内不考核

> 我来给您快速演示一下：

> 我们的合作模式很简单：
>
> 1. 您自己先买一份产品体验（¥3,999/年），确保认可产品质量

**轻度推动**：

> 您考虑得怎么样了？如果确定要加入的话，我这边可以马上帮您开通代理资格，配齐所有销售物料。

> 我理解您的顾虑。其实最简单的验证方式就是：您自己先买一份用用看。
>
> 如果您用下来觉得产品真的好，自然就愿意推荐了；如果觉得不行，那¥3,999也不亏——您自己用一年也完全值回票价了。

**紧迫感推动**：

> 跟您说实话，目前我们这个区域已经有[X]位代理在做了，好的客户资源是有限的。
>
> 如果您能在本周确定，我可以优先把几个主动咨询的客户线索分配给您。
>
> 另外，前10位签约的代理可以享受额外的"首单奖励"——首笔成交额外加5%佣金。

### 3.4 常见拒绝场景应对

**"我再想想"**：
> 没问题，不着急。不过我建议您先体验一下产品——只有用过才知道推荐给客户的时候有没有底气。这样吧，我把免费体验链接发您，您花5分钟试一下？

**"我没有客户资源"**：
> 其实资源是慢慢积累的。您可以先从身边的朋友和同事开始，哪怕推荐1-2个人，也能赚几百块佣金。而且我们会提供推广素材和方法，帮您逐步建立客户群。

**"这个价格客户能接受吗？"**：
> ¥3,999确实不是小数目，但您想想，一个招商方案如果请专业公司来做，至少2-3万。我们的产品能帮客户省90%的成本，而且还能反复使用一年。性价比其实非常高。

**"AI生成的方案客户会信任吗？"**：
> 说实话，一开始很多人都有这个疑虑。但实际用过的客户反馈都很好——因为AI生成的不是那种模板化的空话，而是根据每个展会的具体信息定制的方案。而且客户可以在此基础上修改优化，比从零开始效率高太多了。

## 四、代理商培训SOP

### 4.1 产品培训（2小时）

**第一课：产品认知（60分钟）**
- MeetGrow AI的核心价值主张（为什么客户需要）
- 产品功能全面演示（11模块逐一讲解）
- 6大AI模型的区别和选择建议
- 常见问题和注意事项

**第二课：实操演练（60分钟）**
- 代理商自己操作生成3套方案（不同场景）
- 学会切换不同AI模型
- 熟练掌握导出和分享功能
- 模拟客户Demo演练（角色扮演）

### 4.2 销售技巧培训（2小时）

**第三课：客户开发（60分钟）**

- 客户触达渠道和方法
- 微信群和朋友圈运营技巧
- 客户分类管理（A/B/C/D类）

**第四课：成交转化（60分钟）**
- 5步微信销售序列实操
- 异议处理话术训练
- 报价策略和议价技巧
- 售后跟进和转介绍引导

### 4.3 客户管理培训（1小时）

**第五课：客户成功（60分钟）**

- 常见使用问题解答

## 六、代理商签约流程

### 6.1 完整签约流程（7-14天）

```
Day 1-2: 初次接触 → 发送产品资料和体验链接
Day 3-5: 深度沟通 → 产品演示 + 合作模式讲解
Day 5-7: 犹豫跟进 → 解答疑虑 + 发送成功案例
Day 7-10: 促成签约 → 确认合作意向 + 购买产品
Day 10-14: 启动培训 → 产品培训 + 配发工具包
Day 14+: 正式展业 → 开始推荐客户
```

### 6.2 签约确认方式

由于目前处于早期阶段，采用轻量级签约方式：

1. 微信确认合作意向（文字/语音记录）
2. 代理商购买产品自用（激活码激活）
3. 在代理商管理系统中登记代理信息
4. 发放专属推广码（用于追踪推荐来源）
5. 加入代理商专属微信群

### 6.3 首单支持

为帮助代理商快速建立信心，提供以下首单支持：

- **客户线索分配**：优先分配1-2个主动咨询的客户给新代理
- **陪同Demo**：首次客户演示时，产品负责人可远程协助
- **首单奖励**：首笔成交额外加5%佣金（即铜牌首单佣金为20%）
- **即时结佣**：首单佣金在客户付款后24小时内结算
