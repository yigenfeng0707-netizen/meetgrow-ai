### 批量生成新激活码

如需更多激活码，可通过Python脚本生成：

```python
import random, string

def generate_code():
    seg1 = ''.join(random.choices(string.ascii_uppercase + string.digits, k=4))
    seg2 = ''.join(random.choices(string.ascii_uppercase + string.digits, k=4))
    combined = seg1 + seg2
    h = 0
    for ch in combined:
        h = ((h << 5) - h + ord(ch)) & 0xFFFFFFFF
        if h >= 0x80000000:
            h -= 0x100000000
    h = abs(h)
    checksum = ''
    n = h
    chars = '0123456789abcdefghijklmnopqrstuvwxyz'
    while n > 0:
        checksum = chars[n % 36] + checksum
        n //= 36
    if not checksum:
        checksum = '0'
    checksum = checksum.upper().rjust(4, '0')[:4]
    return f"MG-{seg1}-{seg2}-{checksum}"

# 生成100个
for _ in range(100):
    print(generate_code())
```

## 三、分发策略

### 方式1：直接发文件（推荐起步阶段）

- 通过微信/邮件发送 `MeetGrow_AI.html` 文件
- 配合一份简短的使用说明文档
- 用户免费体验3次后需购买激活码
- 你控制激活码的分发，收到付款后发给对方

**优点：** 零成本、快速启动、完全控制
**适合：** 初期种子用户、个人客户

### 方式2：部署在线网站（推荐规模化阶段）

免费部署平台选择：

| 平台 | 优点 | 步骤 |
|------|------|------|
| GitHub Pages | 完全免费、稳定 | 上传HTML到仓库 → Settings → Pages → 开启 |
| Vercel | 自带域名、CDN加速 | 连接GitHub → 自动部署 |
| Netlify | 拖拽部署、自定义域名 | 拖拽HTML文件到Netlify |
| Cloudflare Pages | 全球CDN、免费SSL | 连接GitHub → 配置构建 |

**部署后的URL示例：**
- GitHub Pages: `https://你的用户名.github.io/meetgrow-ai/`
- Vercel: `https://meetgrow-ai.vercel.app`

**建议：** 购买一个自定义域名（如 `meetgrow.ai`），绑定到部署平台，更显专业。
