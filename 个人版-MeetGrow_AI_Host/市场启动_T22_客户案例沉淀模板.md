## 二、客户成功故事撰写模板

### 2.1 标准格式（500字）

每篇客户成功故事遵循以下结构：

```
标题：[客户名称] 用 MeetGrow AI [解决了什么问题/取得了什么成果]

【客户背景】（约80字）
- 公司/机构名称和简介
- 客户规模和行业
- 使用前的痛点

【面临挑战】（约100字）
- 具体的业务挑战是什么
- 之前是怎么解决的
- 为什么寻求新方案

【解决方案】（约120字）
- 如何了解到MeetGrow AI
- 使用产品的过程
- 主要使用了哪些模块

【使用成果】（约120字）
- 具体的数据变化（效率提升、成本节省等）
- 质量改善的体现
- 客户的主观感受

【客户寄语】（约80字）
- 客户的一句话评价
- 对同行的推荐语

【数据速览】
- 效率提升：X倍
- 时间节省：X小时/X天
- 成本节省：约¥X
- 使用模块：X个
```

### 2.2 撰写原则

1. **数据优先**：每篇至少包含2个具体数据（效率倍数、时间节省、成本节省等）
2. **真实感**：使用客户的原话（加引号），不要过度润色
3. **可验证**：标注客户是否同意公开公司名称（不同意则用化名+行业）
4. **场景化**：让读者能代入自己的场景
5. **简洁**：控制在500字以内，方便在各渠道使用

## 四、NPS评分收集流程

### 4.1 NPS计算方式

NPS = 推荐者占比(%) - 贬损者占比(%)

- **推荐者**（9-10分）：积极推荐，是品牌大使
- **被动者**（7-8分）：满意但不主动推荐
- **贬损者**（0-6分）：可能传播负面口碑

### 4.2 收集流程

```
Day 7  → 发送问卷（含NPS题目）
Day 9  → 未回复提醒
Day 14 → 二次提醒（最后一次）
Day 15 → 汇总统计
月末   → 月度NPS报告
```

### 4.3 NPS跟进策略

**推荐者（9-10分）**：
> 感谢您的高分评价！🎉 您是我们最重要的产品推荐官。如果您身边有同行朋友也有招商方案的需求，帮忙推荐一下？每推荐一位成功购买，您还能获得¥200的推荐奖励。

**被动者（7-8分）**：
> 感谢您的反馈！我注意到您给了[X]分，说明产品整体还不错，但还有改进空间。能具体说说哪方面可以做得更好吗？我们会认真改进。

**贬损者（0-6分）**：
> 感谢您的坦诚反馈。很抱歉产品没有达到您的预期。我想详细了解一下您遇到的问题，看看能不能帮您解决。方便约个10分钟的电话吗？

### 4.4 NPS追踪看板

| 月份 | 问卷发送数 | 回复数 | 回复率 | 推荐者 | 被动者 | 贬损者 | NPS |
|------|-----------|--------|--------|--------|--------|--------|-----|
| | | | | | | | |

**NPS目标**：
- 第1个月：NPS ≥ 30
- 第3个月：NPS ≥ 40
- 第6个月：NPS ≥ 50
