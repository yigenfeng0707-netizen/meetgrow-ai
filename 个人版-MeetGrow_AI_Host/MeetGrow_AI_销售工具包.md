## 二、ROI 计算器

### 2.1 快速ROI计算公式

**公式：** ROI = (节省成本 + 额外收入 - 产品费用) / 产品费用 × 100%

### 2.2 计算示例

**假设条件：**
- 用户：中型活动主办方，年办活动12场
- 产品费用：¥3,999/年
- 每次方案生成API成本：¥0.05

**节省成本项：**

| 项目 | 计算 | 金额 |
|------|------|------|
| 人力节省 | 每场节省2天策划时间 × 12场 × ¥600/天 | ¥14,400 |
| 外包替代 | 6场活动不再外包 × ¥8,000/场 | ¥48,000 |
| 效率提升 | 多版本A/B测试带来的方案优化（按增收5%估算） | ¥5,000 |
| **合计节省** | | **¥67,400** |

**产品成本项：**

| 项目 | 计算 | 金额 |
|------|------|------|
| 产品年费 | | ¥3,999 |
| API调用成本 | 约80次 × ¥0.05 | ¥4 |
| **合计成本** | | **¥4,003** |

**ROI计算：**

```
ROI = (¥67,400 - ¥4,003) / ¥4,003 × 100%
    = 1,584%

即：投入1元，回报16.84元
```
