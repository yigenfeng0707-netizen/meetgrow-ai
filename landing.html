常见问题

API Key 是什么？如何获取？ ▾

API Key 是 AI 服务商提供的访问凭证。MeetGrow AI 支持通义千问、DeepSeek、智谱AI 等 6 家国产模型，您只需在对应平台注册账号即可免费获取。设置页面有详细的获取教程和直达链接。新用户通常都有免费额度，可以零成本试用。

生成一次方案要花多少钱？ ▾

以 DeepSeek 为例，生成一次完整 11 模块方案的费用约 0.01-0.05 元（不到一毛钱）。通义千问和智谱 AI 的新用户免费额度通常可以生成数十次方案。AI 模型的使用成本极低，远低于人工撰写的时间成本。

方案生成不完整怎么办？ ▾

MeetGrow AI 内置了智能续传机制。如果 AI 输出中途停止或遗漏了某些模块，系统会自动检测并发起续传请求，确保最终输出完整的 11 个模块。您无需手动干预。

我的数据安全吗？ ▾

所有数据（API Key、活动信息、方案内容）均保存在您浏览器的本地存储中，不会上传到任何第三方服务器。唯一的外部通信是向您选择的 AI 服务商发送活动信息以生成方案。

支持哪些活动类型？ ▾

MeetGrow AI 适用于大会/峰会、沙龙/论坛、培训会/训练营、园区招商推介会、行业年会等各种活动类型。产品内置了 4 种场景的快速模板，您也可以自定义输入任意活动信息。

可以接入自己部署的大模型吗？ ▾

可以。在设置中选择「自定义」服务商，填入您私有化部署模型的 Base URL、模型名称和 API Key 即可。前提是该模型提供 OpenAI 兼容的 API 接口。
